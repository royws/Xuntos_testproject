# Xuntos_testproject 

Umbraco website. An aplication project for Xuntos

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Installing

A step by step series of examples that tell you how to get a development env running

1. Clone project from github

```
https://github.com/royws/Xuntos_testproject.git
```

2. Clean, Build solution

3. Uncheck LoaderLock Exception Check 

```
Debug>Windows>Exception Settings  Managed Debugging>Assistants>LoaderLock
```

4. Run project on IIS express 

5. Enter name, email, password

6. Customize

7. Choose Microsoft SQL Server Compact (SQL CE)

8. Don't install starter website

9. Go to settings>uSync and do a full import at Import>full import.

10. Reload backoffice to see the newly imported data

11. The front-end should be up and available at the defined localhost:port
